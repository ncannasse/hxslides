Heaps.io

  • 2D + 3D GPU based game framework
  • 2D: Texture, h2d.Sprite, Bitmap, Text etc.
  • 3D: h3d.scene.Object, Camera, Mesh etc.
  • hxd.Res: resource system (abstract file system)
  • Events: mouse keyboard gamepad network
  • HxSL Shader System: write your shaders in Haxe
  • Custom render for advanced effects (PBR etc.) - no fixed pipeline
  • HIDE: Heaps IDE (middleware, data and level editor)
